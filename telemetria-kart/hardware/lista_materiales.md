# Lista de materiales

## Disponibles

- 2 ESP32.
- 2 módulos LoRa RA-01.
- 1 BMI160.
- 1 módulo microSD.
- Powerbank ADATA.
- Cables Dupont.
- Antenas LoRa.
- Capacitor para estabilizar alimentación del LoRa.

## Pendientes

- microSD.
- Regulador 3.3 V o MB102 para alimentar LoRa si siguen los reinicios.
- Caja plástica.
- Soporte o base rígida para fijar la IMU.
- Cables cortos/soldadura para montaje final.
